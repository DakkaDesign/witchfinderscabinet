# 🌿 Witchfinder's Cabinet — Link Hub

Personal link hub for [Witchfinder's Cabinet](https://payhip.com/witchfinderscabinet), a botanical wall art shop.

## Live

👉 [witchfinderscabinet.github.io](https://witchfinderscabinet.github.io/witchfinderscabinet)

## What's inside

A single-page landing site that connects all three storefronts:

| Platform | Format | Link |
|----------|--------|------|
| Payhip   | Digital download (PDF/PNG) | [payhip.com/witchfinderscabinet](https://payhip.com/witchfinderscabinet) |
| Artpal   | Printed & shipped | [artpal.com/witchfinderscabinet](https://www.artpal.com/witchfinderscabinet) |
| Ko-fi    | Support the creator | [ko-fi.com/witchfinderscabinet](https://ko-fi.com/witchfinderscabinet) |

## Structure

```
witchfinderscabinet/
├── index.html       ← main landing page
├── images/          ← product preview images (optional)
├── .gitignore
├── LICENSE
└── README.md
```

## How to update

1. Edit `index.html` directly on GitHub (click the pencil icon)
2. Commit changes — the site updates automatically within ~30 seconds

## Images

Place any product preview images inside the `/images/` folder.
Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
